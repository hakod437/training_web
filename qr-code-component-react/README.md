### What I learned

This project helped me practice layout precision and spacing control in CSS.

Main learning points:

- Difference between `section` and `div` for semantic structure
- Better spacing control with `padding`, `margin`, and `gap`
- When to use `%`, `px`, `rem`, and `ch`
- How to make image sizing responsive inside a card

Example of a CSS pattern I practiced:

```css
.card {
  display: flex;
  flex-direction: column;
  gap: 1rem;
  box-sizing: border-box;
}
```

### Continued development

In future projects, I want to keep improving:
